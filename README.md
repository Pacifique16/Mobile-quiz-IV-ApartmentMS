# Apartment Manager - Quiz Four

Android application for managing apartment listings with SQLite database.

## Features
- Create, Read, Update, Delete (CRUD) operations
- SQLite database for persistent storage
- ListView with custom adapter
- Daily rent calculation
- Edit and Delete functionality for each apartment

## Technologies
- Java
- Android SDK
- SQLite Database
- Material Design Components

## Apartment Model
- Unit Number
- Square Footage
- Rent Amount
- Is AirBnb (Boolean)
- Allows Pets (Boolean)
- Location
